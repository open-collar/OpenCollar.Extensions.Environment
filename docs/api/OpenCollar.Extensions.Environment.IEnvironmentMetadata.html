    Environment

    Gets the functional environment in which the adapter is hosted.

    Declaration
    string Environment { get; }
    Property Value
    Type Description
    System.String

    The functional environment in which the adapter is hosted. null will be returned if the value could not be determined. Environment names are always treated as case insensitive.
    Examples

    E.g. "PDN", "UAT" or "DEV" - this is left up to the implementing application.
