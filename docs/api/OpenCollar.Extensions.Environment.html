    A minimal implementation of the IEnvironmentMetadata interface that can be used a base class.

    EnvironmentMetadataProvider

    A base class from environment metadata providers that implements the basic functionality.

    EnvironmentType

    A class representing an environment to which a resource may belong.

    MismatchedEnvironmentException

    An exception that represents a mismatch in environments (between the application environment and a resource that the application depends upon).

    ServiceCollectionFluentExtensions

    Extensions to the Microsoft.Extensions.DependencyInjection.IServiceCollection type allowing the application service and related services to be registered.
